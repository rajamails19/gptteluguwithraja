# Vercel deployment

This app is a TanStack Start app, so it should deploy as a server-rendered app rather than as a static SPA.

## Project settings

- Install command: `npm install`
- Build command: `npm run build`
- Output: generated by Nitro in `.vercel/output`

## Environment variables

Set these in Vercel if you want Telugu read-aloud audio:

- `SARVAM_API_KEY`
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`

The site can still render without these variables, but `/api/tts` will return a configuration error until `SARVAM_API_KEY` is set.

Supabase client/auth helpers also expect these if you start using client-side Supabase features:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
